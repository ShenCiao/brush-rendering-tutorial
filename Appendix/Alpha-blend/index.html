Alpha Blending

Previously, we touched upon alpha compositing formula in the Vanilla -> Opacity section. In this section, we delve deeper to offer a more detailed derivation of alpha blending.

Compositing Algebra​

Let's define the color of an object as a=(Ca,Aa)a=(C_a,A_a)a=(Ca​,Aa​), where CaC_aCa​ represents the RGB color components, AaA_aAa​ represents the alpha value. If the object is opaque (Aa=1A_a=1Aa​=1), the color is represented by a^\widehat{a}a. Otherwise when the object has variable transparency (Ax≠1A_x \neq 1Ax​=1), the color is simply represented by aaa.

We use the notation a⊙ba \odot ba⊙b to denote the alpha blending of colors aaa and bbb. The associative property of blending dictates:

(a⊙b)⊙c=a⊙(b⊙c)(a \odot b) \odot c = a \odot (b \odot c)(a⊙b)⊙c=a⊙(b⊙c)

In scenarios where a foreground object with color xxx, is blended over an opaque background with color y^\widehat{y}y​, the resulting color o^\widehat{o}o can be expressed as:

o^=x⊙y^\widehat{o} = x \odot \widehat{y}o=x⊙y​

And CoC_oCo​ is given by:

Co=AxCx+(1−Ax)Cy(1)C_o = A_xC_x + (1 - A_x)C_y \tag{1}Co​=Ax​Cx​+(1−Ax​)Cy​(1)

To calculate the output color ooo by alpha blending two colors aaa and bbb, o=a⊙bo = a \odot bo=a⊙b, where aaa and bbb both have variable opacity, we further blend the color with an opaque background c^\widehat{c}c, which leads to: o⊙c^=a⊙(b⊙c^)o \odot \widehat{c} = a \odot (b \odot \widehat{c})o⊙c=a⊙(b⊙c)

Substituting both sides with Equation (1)(1)(1) gives us:

AoCo+(1−Ao)Cc=a⊙(AbCb+(1−Ab)Cc)=AaCa+(1−Aa)(AbCb+(1−Ab)Cc)=AaCa+(1−Aa)AbCb+(1−Aa)(1−Ab)Cc\begin{aligned} A_oC_o+(1-A_o)C_c &=a \odot(A_bC_b+(1-A_b)C_c) \\&=A_aC_a+(1-A_a)(A_bC_b+(1-A_b)C_c) \\&=A_aC_a+(1-A_a)A_bC_b+(1-A_a)(1-A_b)C_c \end{aligned}Ao​Co​+(1−Ao​)Cc​​=a⊙(Ab​Cb​+(1−Ab​)Cc​)=Aa​Ca​+(1−Aa​)(Ab​Cb​+(1−Ab​)Cc​)=Aa​Ca​+(1−Aa​)Ab​Cb​+(1−Aa​)(1−Ab​)Cc​​

This leads to an equation of the form X0+Y0Cc=X1+Y1CcX_0+Y_0C_c=X_1+Y_1C_cX0​+Y0​Cc​=X1​+Y1​Cc​.

By setting X0=X1X_0=X_1X0​=X1​ and Y0=Y1Y_0=Y_1Y0​=Y1​, we can get:

Co=AaCa+(1−Aa)AbCbAoC_o=\frac{A_aC_a+(1-A_a)A_bC_b}{A_o}Co​=Ao​Aa​Ca​+(1−Aa​)Ab​Cb​​ Ao=1−(1−Aa)(1−Ab)A_o=1-(1-A_a)(1-A_b)Ao​=1−(1−Aa​)(1−Ab​)

Transparency​

Opacity is the measure of the extent to which an object blocks light, while transparency is its inverse, representing the object's see-through quality. We commonly use the alpha values to represent opacity. But sometimes, replacing the opacity with transparency can make life easier. Consider the alpha value of a pixel, A∈[0.0,1.0]A \in [0.0, 1.0]A∈[0.0,1.0]. The transparency TTT of this pixel is:

T=1−A(2)T = 1 - A \tag{2}T=1−A(2)

To calculate the result transparency ToT_oTo​ by composing two objects with transparencies TaT_aTa​ and TbT_bTb​ separately, we can simply multiply them: To=Ta×TbT_o = T_a \times T_bTo​=Ta​×Tb​ To transition from transparency to opacity, we substitute with Equation (2)(2)(2), arriving at:

1−Ao=(1−Aa)(1−Ab)1 - A_o = (1 - A_a)(1 - A_b)1−Ao​=(1−Aa​)(1−Ab​) Ao=1−(1−Aa)(1−Ab)A_o = 1 - (1 - A_a)(1 - A_b)Ao​=1−(1−Aa​)(1−Ab​)

As for the composition of multiple transparencies, it is a straightforward multiplication of the individual transparencies:

To=∏i=1nTiT_o=\prod_{i=1}^n{T_i}To​=i=1∏n​Ti​

Similar to previous deriviation, we can get:

Ao=1−∏i=1n(1−Ai)A_o=1-\prod_{i=1}^n{(1-A_i)}Ao​=1−i=1∏n​(1−Ai​)
